# 曼旮

曼旮（manga）是一个浏览器内使用的javascript阿卜（app），供漫画提供者以及收藏者使用。

以防黑客利用，浏览器不允许javascript与其他服务器通话，所以阿卜本身的文件需与漫画文件在同一服务器。

再有，浏览器一般不允许javascript抽取浏览器所在电脑的文件。所以收藏者需找一个虚拟本地服务器的软件。
